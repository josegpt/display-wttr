* Installation
** Manual
Download =display-wttr.el= to your local directory.
*** Git
#+BEGIN_SRC shell
  git clone https://github.com/josegpt/display-wttr
#+END_SRC
*** Curl
#+BEGIN_SRC shell
  curl -s -o display-wttr.el https://raw.githubusercontent.com/josegpt/display-wttr/main/display-wttr.el
#+END_SRC
